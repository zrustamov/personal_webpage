Haqqımda

AI sistemləri haqqında necə düşünürəm

Tətbiqi AI-da maraqlı problem nadir hallarda modeldir. O modelin ətrafındakı hər şeydir — data pipeline-ları, retrieval, qiymətləndirmə, heç kimin sizi xəbərdar etmədiyi failure mode-lar.

MegaSec AI Laboratoriyasında Süni İntellekt mühəndisi və ADA Universitetinin nəzdində olan Data Analitikası və Tədqiqat Mərkəzində tədqiqatçıyam. Vaxtımın bir hissəsini nazirliklərə gedən hüquqi AI sistemləri qurmağa, digər hissəsini isə bu sistemləri real sorğu paylanmaları altında dayanıqlı saxlayan metodları öyrənməyə sərf edirəm.

İşlərimin əksəriyyəti Azərbaycan dili ilə bağlıdır — məhdud NLP resursları olan dil. Bu vəziyyət ingilisdilli AI ədəbiyyatının optimizə etdiyi tradeoff-lardan fərqli bir trade-off setini tələb edir. Bu məhdudiyyət karyeramda ən məhsuldarlardan biri olub.

Nə üzərində işləyirəm

  • Retrieval-Augmented Generation — hibrid dense və sparse indekslər, fusion strategiyaları, LLM reranking. Domen dəyişikliklərinə davamlı RAG.
  • Hüquqi AI — qanunverici korpuslar üçün retrieval və əsaslandırılmış cavab sistemləri, ilk addım Azərbaycan qanunvericiliyi.
  • Çox-agent arxitekturalar — intent yönləndirmə, ixtisaslaşmış agentlər, agent keyfiyyəti üçün qiymətləndirmə çərçivələri.
  • Az-resurslu Azərbaycan NLP — embedding-lər, orfoqrafik düzəliş, sentiment analizi.

Necə işləyirəm

Əvvəlcə qiymətləndirmə setini yazıram. Modelə toxunmadan əvvəl bir domen üçün yaxşı retrieval-ın necə göründüyünü təyin edə bilmirəmsə, layihə dreyf edəcək. Eval yerindədirsə, qalanı əsasən ölçmədir.

Anlaya bilməyəcəyim ağıllı arxitekturalardansa, gecə 2-də debug edə biləcəyim sadə arxitekturalara üstünlük verirəm. Production sistemlər zamanla pisləşir; ən uzun yaşayan kod altı ay sonra oxuya biləcəyim koddur.

Tədqiqat və mühəndislik mənim üçün ayrı yollar deyil. Çap etdiyim məqalələr göndərmə zamanı qarşılaşdığım problemlərdən doğub — göndərdiyim sistemlər də əvvəlcə oxumalı olduğum metodlardan faydalanıb.

İstifadə etdiyim alətlər

Python FastAPI Milvus BGE-M3 Docker Linux QLoRA / PEFT Git / CI

İşdən kənar

Retrieval metodları, production AI-da mühəndislik tradeoff-ları və az-resurslu dil üçün NLP qurmağın necə olduğu haqqında zaman-zaman yazıram. Bu sahədə işləyirsinizsə, eşitməkdən məmnun olaram.
